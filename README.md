# 🛒 EMarket - E-Commerce Microservices Application

> 🚧 **Project Status:** This project is currently under active development. New features and microservices are being implemented continuously. The architecture, APIs, and documentation may change as the project evolves.

## Overview

EMarket is a backend e-commerce application built using **Java** and **Spring Boot** following a **Microservices Architecture**. The project aims to simulate a real-world online shopping platform by separating core business functionalities into independent microservices such as User, Seller, Product, Cart, and Order services.

The primary goal of this project is to learn and implement modern backend development concepts, including RESTful APIs, distributed system design, service communication, cloud databases, and scalable application architecture.

## 🚀 Planned Features

- ✅ User Service
- ✅ Product Service
- ✅ Order Service
- ⏳ Seller Service (In Progress)
- ⏳ Cart Service (In Progress)
- ⏳ Service Discovery (Eureka)
